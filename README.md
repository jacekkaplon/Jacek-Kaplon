## 🔍 Featured Project — The Little Fish Hunter

A home SOC environment built to detect, analyse, and document real-world threats.

- 9 Wazuh SIEM agents across 5 OS types on Proxmox
- Live Cowrie SSH honeypot on isolated VLAN 99 (Raspberry Pi)
- Multi-hop WireGuard VPN with GRETAP L2 overlay
- Custom XML detection rules for non-standard log sources (Technitium DNS, Cowrie)
- VirusTotal REST API integration for automated IOC enrichment
- Structured IR tickets mapped to MITRE ATT&CK
